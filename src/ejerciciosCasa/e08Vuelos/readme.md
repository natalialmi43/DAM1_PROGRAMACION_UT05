# UT5 - Ejercicios 8 - Vuelos


Tenemos una estructura donde se almacenan ventas de vuelos. Tiene el siguiente formato:

Map<String, Map<String, Double>> vuelosPasajerosCoste = new HashMap<>();

Aquí, están los códigos de vuelo como clave del hashmap exterior, y dentro los DNI de los pasajeros con el coste del billete asociado.

Se pide:
Generar un set con los identificadores de los vuelos
Mostrar el precio del billete más caro
Crear un Map que la clave sea el vuelo y el valor el precio medio del billete
Crear un MAP que la clave sea el DNI y el valor el precio que se ha gastado en billetes

Bola extra. Utiliza los métodos vistos en las clases auxiliares (Arrays y Collections) para realizar las ordenaciones donde sea necesario.
